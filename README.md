# Breast-Cancer-22-Gene-Signature
Integrative transcriptomic analysis identifying a novel 22-gene prognostic signature associated with the proteasome-chemokine axis in breast cancer progression.


"Note: Please download the clinical data from GSE96058 or ensure phenodata_clean.csv is in your working directory before running the script."




External Validation Summary:Dataset: TCGA-BRCA ($n=1094$)KM Plot: Significant separation between High-risk and Low-risk groups ($p < 0.0001$).Accuracy: Time-dependent AUC for 1, 3, and 5 years shows high predictive performance.C-index: 0.76
